<h1>Landing Page — LAMANSH</h1>

<p>
  Це односторінковий лендінг, виконаний за макетом зі стокового дизайну Figma.
  Проєкт реалізовано з використанням HTML, CSS, JavaScript та анімаційної бібліотеки GSAP.
</p>

<h2>Демо версія</h2>
<p>
  <a href="https://store-wedding-dresses.netlify.app/" target="_blank">Переглянути демо</a>
</p>

<h2>Функціонал проєкту</h2>
<ul>
  <li>Адаптивна верстка для мобільних, планшетів та десктопів.</li>
  <li>Плавні анімації елементів під час прокрутки (GSAP + ScrollTrigger).</li>
  <li>Інтерактивна форма з базовою валідацією на JavaScript.</li>
  <li>Кнопка "Повернутися нагору".</li>
  <li>Ховер-ефекти та плавні переходи.</li>
</ul>

<h2>Технології</h2>
<ul>
  <li>HTML5</li>
  <li>CSS3 (включно з адаптивом та анімаціями)</li>
  <li>JavaScript</li>
  <li>GSAP</li>
</ul>

<h2>Особливості</h2>
<ul>
  <li>Дизайн використано лише як навчальний макет, без комерційного призначення.</li>
  <li>Форма працює на стороні клієнта: перевіряє заповнення полів, але не надсилає дані на сервер.</li>
  <li>Код структурується відповідно до принципів чистої та зрозумілої верстки.</li>
</ul>

<h2>Як використовувати</h2>
<ol>
  <li>Склонуйте репозиторій або завантажте файли.</li>
  <li>Відкрийте файл <strong>index.html</strong> у браузері.</li>
  <li>Сайт працює локально без додаткових налаштувань.</li>
</ol>
